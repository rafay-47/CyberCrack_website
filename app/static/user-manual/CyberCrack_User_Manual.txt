4. Antivirus Exclusion Steps
═══════════════════════════════════════════════════════════════════

Adding a File to Exclusions in Windows Security and AVG
1. Windows Security (Windows Defender)
Open Windows Security:
- Press ⊞ Win, type 'Windows Security', and press Enter.
Go to Virus & threat protection:
- In the left‑hand menu, click 'Virus & threat protection'.
Manage settings:
- Under 'Virus & threat protection settings', click 'Manage settings'.
Open Exclusions settings:
- Scroll down to 'Exclusions' and click 'Add or remove exclusions'.
Add a new exclusion:
- Click 'Add an exclusion' ► 'File'. Browse to and select the file you want to exclude, then click 'Open'.
Verify your exclusion:
- You’ll see the file listed. Close the Windows Security window.

2. AVG Antivirus
Open the AVG user interface:
- Double‑click the AVG icon in your system tray or open AVG from the Start menu.
Access Settings:
- Click the Menu (☰) in the top‑right corner and choose 'Settings'.
Navigate to Exceptions:
- In the left pane, select 'General' ► 'Exceptions'.
Add a new exception:
- Click 'Add Exception'. Browse to and locate the file you wish to exclude, select it, and click 'OK'.
Save your changes:
- Ensure the new exception appears, then click 'Add exception' (or 'OK') to confirm.
Close AVG Settings:
- Exit the Settings window; AVG will now ignore the file.

7. BUTTON REFERENCE
═══════════════════════════════════════════════════════════════════

🗑️ CLEAR TRANSCRIPT (C)
• Clears all transcribed text from your responses area
• Hotkey: Press 'C' key
• Preserves AI feedback and screenshot analysis
• Resets transcript for fresh input without losing AI responses

⏸️ FREEZE (F)
• Pauses all AI processing temporarily
• Hotkey: Press 'F' key
• Changes to "▶️ Unfreeze (F)" when active
• Useful when you need to think without AI interference
• Automatically freezes after screenshot analysis

🎧 INTERVIEW MODE (I)
• Toggles between normal and interview-optimized mode
• Hotkey: Press 'I' key
• Normal: "🎧 Start Listening (I)" - captures all audio
• Active: "🔴 Interview Mode (I)" - optimized for speaker audio
• Interview mode focuses on capturing interviewer questions

📷 SCREENSHOT
• Captures full screen and analyzes code/technical content
• Hotkey: Press 'S' key
• Temporarily hides application window for clean capture
• Provides AI analysis of code, security vulnerabilities, and improvements
• Automatically freezes display after analysis to review results

🔝 ALWAYS ON TOP
• Toggle switch to keep application window above all others
• Enabled by default for seamless interview workflow
• Useful during video interviews or while coding
• Can be toggled on/off as needed

🔑 ACTIVATE
• License activation button
• Enter license key in the text field and click to activate
• Manages trial period and full license access
• Required for continued use after trial expires

11. TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

COMMON ISSUES AND SOLUTIONS:

🎤 NO AUDIO TRANSCRIPTION:
• Check microphone permissions in Windows/macOS settings
• Verify default recording device in Sound settings
• Test microphone in other applications
• Restart the application
• Check if FFmpeg is installed and accessible

❌ APPLICATION WON'T START:
• Run as Administrator (Windows)
• Check if all dependencies are installed
• Verify Windows Defender isn't blocking the executable

🔄 SLOW PERFORMANCE:
• Close other resource-intensive applications
• Check internet connection for AI features
• Consider using a computer with more RAM

🌐 AI RESPONSES NOT WORKING:
• Check internet connection
• Verify license status
• Try restarting the application
• Check if antivirus is blocking network access
• Ensure firewall allows the application

📱 SCREENSHOT ANALYSIS FAILED:
• Try taking the screenshot again
• Ensure the screen content is clearly visible
• Check if image contains readable text/code
• Verify internet connection for AI analysis
• Close other applications that might interfere
